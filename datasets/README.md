# Data Acquisition

## Prerequisites

This project uses Azure SQL Edge running in Docker. Before running the SQL scripts:

1. Install Docker Desktop for Mac: https://docker.com/products/docker-desktop
2. Install Azure Data Studio: https://aka.ms/azuredatastudio
3. From the project root, run: `docker-compose up -d`
4. Wait 15 seconds, then connect via Azure Data Studio to `localhost,1433` with user `sa` and password `NfipWarehouse2026!`

Alternatively, use the helper: `./scripts/docker_start.sh`

---

The CSV data files in this directory are generated by the Python ingestion scripts and are NOT committed to the repository due to file size.

## How to acquire the data

1. Create a Python virtual environment: python -m venv venv
2. Activate it: source venv/bin/activate (Mac/Linux) or venv\Scripts\activate (Windows)
3. Install dependencies: pip install -r requirements.txt
4. Run the ingestion scripts:
   - python scripts/ingest/fetch_nfip_claims.py
   - python scripts/ingest/fetch_nfip_policies.py

This will download NFIP flood insurance data from the OpenFEMA API for 5 target states (FL, LA, TX, NJ, NY) and save CSV files into datasets/claims/ and datasets/policies/.

## Data Source
- API: https://www.fema.gov/api/open/v2/FimaNfipClaims
- API: https://www.fema.gov/api/open/v2/FimaNfipPolicies
- Documentation: https://www.fema.gov/about/openfema/api
- No API key required. Free and open access.
